JQuery horizontal dropdown menu including left and right sub-menu deployment

The dropdown menu is built from multi-level unordered lists. Each one of these corresponds to a deployed sub-menu, rendered as follows:

  • Level 1:
    A horizontal list of items (the main menu).
  • Level 2:
    A vertical list of items deployed below a level 1 item.
  • Level n > 2:
    A vertical list of items deployed to the left or right of item n-1.
A dropdown sub-menu has the following structure: 
    <li class="dropdown"><a href="#">Anchor text</a>
        <ul class="sub-menu">
            <li>
            
            ...
            
            </li>
        </ul>
    </li>
For left deployment of a sub-menu, add class "left" to the ul tag.
